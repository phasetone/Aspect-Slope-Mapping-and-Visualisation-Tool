# Aspect-Slope-Mapping-and-Visualisation-Tool
All-in-one aspect-slope mapping and visualisation tool for QGIS developed using PYQGIS 3.

Useful to quickly output aspect, slope and aspect-slope maps with a visualisation style highlighting a priority aspect orientation from an input DEM.
Please use the example .tif file as the input raster for a demonstration of the tool's function.
